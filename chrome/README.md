# Local Home SDK Chrome Testing Environment

This is a Local Home SDK implementation that runs in the Chrome browser.

# Usage

**To compile the app:**:

- `npm run install`
- `npm run build`

To run the app, you'll need a couple shell instances:

**1. Start the radio proxy server:**

- `npm run proxy` to start the node proxy server.

**2. Start the chrome client:**

- `npm run host` to serve the chrome client.
- Open `localhost:4321` in your chrome.
